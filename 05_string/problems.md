# String Problems

## Basic Problems

### 1. Find length of string

```bash
Enter a string: akkal
Length of a string: 5
```

### 2. Find reverse of string

```bash
Enter a string: Akkal Dhami
Reversed string: imahD lakkA
```

### 3. Count no. of alphabets, vowel alphabets and consonant alphabets in a string

```bash
Enter a string: Akkal

No. of alphabets: 5
No. of vowels: 2
No. of consonants: 3
```

### 4. Find the given string is uppercase, lowercase or digit and print the ASCII code of each string

```bash
Enter a string: aA1d

a is a lowercase string.
ASCII Value of a : 97

A is an uppercase string.
ASCII Value of A : 65

1 is a digit.
ASCII Value of 1 : 49

d is a lowercase string.
ASCII Value of d : 100
```

### 6. Convert lowercase string to uppercase and vice-versa

```bash
Enter a string: AkkAL

Lowercase string: akkal
Uppercase string: AKKAL
```

```bash
Enter a string: AkkAL

```

